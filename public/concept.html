Core architecture

  • API layer validates request and policy
  • Adapter selects model/provider
  • Response sanitizer enforces output rules
  • Observability logs latency, tokens, and failures

Guardrail model

  • Input moderation
  • Prompt injection screening
  • Output filtering and schema checks
  • Rate and quota controls
